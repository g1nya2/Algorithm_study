터미널에서 작성법:
```bash
git commit -m "[BOJ] Hello World" -m "https://www.acmicpc.net/problem/2557"
```
+ 플랫폼 작성법 통일:
  + [BOJ] - 백준
  + [PGS] - 프로그래머스
